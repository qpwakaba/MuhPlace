【プラグイン名】
　　MushPlace

【なにこれ】
　　きのこや苗木などの装飾系ブロックを菌糸や土だけでなく様々な場所に設置できるようにするプラグインです。

【ライセンスとか免責事項とかそういうのは】
　　ライセンスはよくわからないので特に指定しませんが、
　　再配布、改変して配布 は許可がない場合は基本的に禁止とさせていただきます。
　　デコンパイルや、改変しての利用はどうぞ好き勝手にやっちゃってください。
　　一応ソースコードも入っているので見てみてください。(正式版までは付属させます)
　　（どこを改変して使ってるか教えてくれると嬉しいなぁ〜ﾁﾗｯ）

　　またこのプラグインを使用したことによる損害(ワールドがぶっ壊れたとか)は
　　責任は取りません(ってか取れません)ので、自己責任での使用をお願いします。

【対応バージョンは？】
　　自分でも把握していないんですが、1.6.4、1.7.2で動作確認しています。
　　動いたよー、動かなかったよーって教えてくれるとありがたいです。

【どうやって使うの】
　　・導入方法
　　　　pluginsフォルダーにコピーするだけで動作します。

　　・使い方
　　　　置きたいアイテムを手に持って右クリックするだけ。
　　　　普段ブロックを設置する時と同じような感覚で置くことができます。
　　　　
　　　　アイテムのデータ値にも対応しているので
　　　　手に持っているアイテムのデータ値を保持したまま
　　　　ブロックとして設置することができます。
